\begin{frame}{Since mid-April: Flutter project work}
    \begin{itemize}
        \item Built the transport around cryptographic identity: Ed25519 public keys as peer identity and signed packets as the basic trust object.
        \item Implemented BLE discovery and nearby messaging, including configurable BLE central/peripheral roles for device testing.
        \item Added UDP/UDX connectivity plumbing for longer-range friend communication and an Internet-scale baseline.
        \item Added NAT traversal support: public address discovery, well-connected friend facilitators, hole-punching state, reconnect signaling, and rendezvous coordination.
        \item Added friendship state in Redux: pending/accepted friendships, persistence, unfriend handling, friends-of-friends, and friend-shared rendezvous servers.
        \item Added Noise sessions for hop-level channel security between connected peers.
        \item Added delivery machinery: ACK/read receipts, fragmentation/reassembly for larger payloads, duplicate suppression, and routing tests.
    \end{itemize}
\end{frame}
