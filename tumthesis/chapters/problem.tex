\section{Design Goals}
\begin{itemize}
    \item Prefer local over remote communication
    \item When available, use the Internet-scale relation to relay messages over long distances
    \item Reduce attack interface for surveillance and monitoring as much as possible: expose the minimum amount of required information to facilitate communication
    \item Store-carry-forward: cache packets for unavailable recipients until a possible delivery attempt can be made
    \item Use existing mobile infrastructure to facilitate Internet-scale communication: edge rendezvous points and hole-punching
\end{itemize}
